CROM Diff Viewer

O(1) Delta Extraction

Visão do Engenheiro

O Git usa Diff Textual (Myers Algorithm), lendo linhas alteradas e salvando o delta inteiro. Ouve muito overhead em repositórios massivos.

Solução Crompressor

O CROM usa LSH (Locality-Sensitive Hashing). Ele vê blocos de texto não como caracteres, mas como "IDs em uma B-Tree".

Se você copia um método inteiro para outro arquivo, o CROM descobre na hora que o Hash ID já existe e custa 0 bytes a mais.

Instrução: Edite os painéis ao lado. As curvas de Bézier vermelhas mostram os Deltas gerados em tempo real matematicamente. As opacidades revelam a deduplicação.
