# Crompressor Web Ecosystem 🧬

**O Frontend de Demonstração Interativa do Motor Semântico Crompressor.**

Este repositório serve como a vitrine tecnológica executável em WebAssembly (WASM) do motor de compressão O(1) escrito em Go puro. Ele espelha fisicamente os conceitos descritos no repositório backend, demonstrando a superioridade da pesquisa e desenvolvimento em VFS e Neural Codebooks.

## 🚀 O Portal (The Hub)
Acesse `/hub/index.html` para ver o painel consolidado com todos os links executáveis.

## 🧠 Core Tecnológico Provado
- O núcleo computacional reside na pasta `12-crom-wasm-core` onde o binário do Crompressor Go foi instanciado como wasm_exec.
- Todos os arquivos e cliques transitam pelo barramento universal da `crom-bridge.js`, provando a execução *Lossless* mecânica.
